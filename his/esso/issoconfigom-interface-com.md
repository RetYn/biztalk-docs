# ISSOConfigOM Interface (COM)
The **ISSOConfigOM** grants access to the Single Sign-On (SSO) object model for server configuration.  
  
## Requirements  
 **Type Library**: SSOConfigOM 1.0 Type Library (SSOConfigOM.dll)
